### 🛡️ Le projet

**CyberAudit & Solutions** est une **startup en cybersécurité** que j'ai co-fondée avec deux autres étudiants de Dijon dans le cadre des **« Les Entrep' »**, une association entrepreneuriale qui accompagne les étudiants dans la création de leur entreprise.

Dans le cadre du **projet portfolio de Holberton School**, nous concevons une **plateforme web full-stack** destinée aux **PME ne disposant pas de compétences IT en interne**.

### 🎯 Services proposés par la startup

| Service | Description |
|---|---|
| 🔍 **Audit & analyse des risques** | Méthodologie **EBIOS RM** (référentiel ANSSI) |
| 🚨 **Gestion d'incidents** | Remédiation des vulnérabilités |
| 🛡️ **Protection & SOC 24/7** | Supervision continue des systèmes |
| 🎓 **Sensibilisation** | Formation des équipes clientes |

### 💡 Fonctionnalités de la plateforme

- 📦 **Choix d'un pack de service** parmi les 4 offres CyberAudit
- 📋 **Formulaire structuré** pour soumettre une demande
- 📧 **Confirmation par email automatique**
- 📊 **Suivi en temps réel** de l'avancement du dossier
- 📑 **Rapports clairs de vulnérabilités**
- 🎓 **Accès aux modules de sensibilisation**
- 🔐 **Contrôle d'accès basé sur les rôles (RBAC)** + authentification **JWT**
